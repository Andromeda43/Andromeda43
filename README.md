## What I'm working on

**SciText-ES** — My thesis. NLP pipeline for rhetorical segmentation of Spanish academic papers. Fine-tuned SciBETO-large on a 1.8M-document corpus from CORE API; also benchmarked open-weight LLMs vs GPT/Gemini on zero-shot prompting. Deployed on AWS ECS.  
→ [github.com/jdyepest/entrega_2_micropryecto](https://github.com/jdyepest/entrega_2_micropryecto)

**GoAct** — A small experiment in Go implementing the CodeAct paradigm: the LLM writes executable bash/Go code as its action space instead of JSON tool calls. Runs each block, feeds output back, iterates.  
→ [Andromeda43/MLTest-GoAct](https://github.com/Andromeda43/MLTest-GoAct)
